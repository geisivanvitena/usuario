# ms-usuarios

Microserviço responsável pelo gerenciamento de usuários do sistema, incluindo cadastro, autenticação e autorização.

## Responsabilidades
- Cadastro e manutenção de usuários
- Autenticação via JWT
- Controle de perfis e permissões

## Tecnologias
- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- Spring Security + JWT
- Gradle
- Banco de Dados: PostgreSQL
